SDK 及相关资源下载

iOS

Objective-C SDK 包含数据存储、即时通讯与推送、社交、用户反馈、应用内搜索等模块。Swift SDK 包含数据存储、即时通讯与推送等功能。

  • Objective-C SDK
  • Swift SDK
  • 用户反馈组件

JavaScript

根据功能需求选择对应的 SDK

  • 数据存储 SDK
  • 即时通讯 SDK

Java

包含 LeanCloud 平台全部功能的客户端接口，适用于 Java 和 Android 两个平台。SDK 中所有 API 接口与 LeanCloud 云端交互严格遵循 LeanCloud REST API 规范，并且全部采用了 RxJava 风格来设计。

  • Java SDK

Unity

自 2016 年 12 月起，原来的 AVOSCloud.dll 按照模块被拆分为 Core、Storage、Analytics、Push 等不同的文件。阅读 SDK 安装指南 获取帮助，或者访问 Unity SDK Release 查看所有历史版本。

  • Unity SDK

.NET（新版）

自 2016 年 12 月起，.NET SDK 将统一代码并且按照模块被拆分为 Core、Storage、Analytics、Push 等不同的文件。阅读 SDK 安装指南 获取帮助，或者前往 Nuget 进行安装。

  • .NET SDK

其他 SDK（开放源码）

包括主要用于服务端的 Python、PHP SDK，以及旧版本的 Android、Java SDK. 可参考 SDK 安装指南 了解安装方法。

  • Android SDK（旧版）
  • Java SDK（旧版）
  • Python SDK
  • PHP SDK
